# 永平工商會議室快速預約系統

## 專案簡介
本專案為「永平工商會議室快速預約系統」，提供校內教職員工快速建立 Google 日曆會議室活動連結，並自動檢查會議室時間衝突，提升預約效率。

## 主要功能
- 輸入處室單位、活動名稱、選擇會議室、日期與時間
- 自動產生 Google 日曆活動連結，並邀請會議室資源
- 會議室時間自動衝突檢查（即時查詢 Google Calendar 公開 iCal）
- 活動標題自動格式化為「【處室單位】活動名稱」
- 支援複製連結與自動開啟 Google 帳戶登入
- 介面美觀，支援行動裝置

## 檔案結構
- `index.html`：主網頁，表單與說明
- `script.js`：主要功能邏輯（表單驗證、日曆連結產生、衝突檢查等）
- `styles.css`：自訂美觀樣式
- `會議室對應公開網址.txt`：各會議室的 Google 日曆公開網址與 iCal 連結
- `會議室清單.txt`：各會議室名稱與對應 Google Calendar 資源郵件
- `網址編譯範例.txt`：Google 日曆活動連結編譯過程紀錄

## 使用說明
1. 開啟 `index.html`
2. 依序填寫處室單位、活動名稱、選擇會議室、日期與時間
3. 點擊「產生日曆連結」
4. 系統自動檢查會議室時間衝突，無衝突則自動複製連結並開啟 Google 日曆活動建立頁面
5. 登入 Google 帳戶後即可完成預約

## 注意事項
- 會議室衝突檢查需網路連線，若遇到 CORS 問題請手動確認會議室可用性
- 本系統僅供永平工商校內使用
